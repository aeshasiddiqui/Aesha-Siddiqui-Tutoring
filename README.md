# Aesha Siddiqui Tutoring Website

This folder contains a simple tutoring website for Aesha Siddiqui Tutoring. It is a small website made from three main files:

- `index.html` controls the words and page sections.
- `styles.css` controls the colors, spacing, layout, and overall look.
- `script.js` controls small interactive parts of the page.

You do not need a special app account or an online service to view the website while editing it.

## What is on the website

The page includes:

1. A home section with a short introduction.
2. A services section for tutoring options.
3. An about section explaining the tutoring approach.
4. A step-by-step process section.
5. A services and rates section with package options.
6. A referral section.
7. A testimonials and success stories section.
8. A scheduling section with a sample calendar.
9. A website tools section.
10. A contact section with a consultation request form.
11. A navigation menu at the top of the page.

## How to open the website

The easiest way to view the site is with Visual Studio Code and the Live Server extension.

1. Open this folder in Visual Studio Code.
2. Find `index.html` in the file list.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
5. Your browser should open the website.

If you do not have Live Server installed:

1. Open Visual Studio Code.
2. Click the Extensions button on the left side.
3. Search for **Live Server**.
4. Install the extension named **Live Server**.
5. Follow the steps above again.

## How to make simple text changes

Most words shown on the website are in `index.html`.

1. Open `index.html`.
2. Find the words you want to change.
3. Edit the text carefully.
4. Save the file.
5. Refresh the browser, or wait for Live Server to update the page.

For example, to change the email address shown on the page, look for:

```html
hello@example.com
```

Then replace it with the correct email address.

## How to change colors or spacing

Most visual styling is in `styles.css`.

Use this file when you want to change things like:

- Background colors
- Button colors
- Text sizes
- Spacing between sections
- How the page looks on phones or computers

After editing `styles.css`, save the file and check the page in your browser.

## How to check your changes

After making a change:

1. Open the website in your browser.
2. Check the part of the page you changed.
3. Try the menu links at the top.
4. Make sure the page still looks good on a smaller screen, such as a phone-sized browser window.
5. If you changed the contact form, type into the form and click **Send Consultation Request** to make sure the message area still responds.
